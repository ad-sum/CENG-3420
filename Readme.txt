============ File Description ============
ALU32.vhd               -> 32bit adder component
components_package.vhd  -> a package of all components
const_package.vhd       -> define constant values used by other programs
controller.vhd          -> a controller, decide what to do given the instruction
instruction_fetch.vhd   -> a simple fetcher
processor_core.vhd      -> implemented the processor core
sign_extend             -> extend from 16 bits to 32 bits
